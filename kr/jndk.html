메인 화면

하이엔드 제품들로 제품 홍보 이미지를 사용하여 전반적인 디자인의 흐름을 잡았습니다. 메인 콘텐츠 부분의 왼쪽은 제품 상세 내용과 오른쪽은 제품 이미지를 사용하였습니다.

  • 레이아웃

    헤더에는 로고와 메인 메뉴를 배치, 메인 콘텐츠 왼쪽은 제품 상세와 오른쪽은 브랜드 관련 서브메뉴와 제품 이미지를 배치하였습니다. 푸터 영역은 구독과 CS(고객센터)를 넣었습니다. 헤더와 푸터는 모든 화면에 공통으로 사용됩니다.

  • 메인 화면 레이아웃

제품 화면

가장 중요한 부분인 제품 섬네일과 가격의 사이즈는 키웠습니다. 최대 다섯 개까지 제품이 디스플레이됩니다.

  • 레이아웃

    공통으로 사용되는 헤더의 메뉴 > 서브 메뉴 > 서브 메뉴 타이틀 버튼으로 계단식으로 배치하였으며, 타이틀에 알맞은 콘텐츠는 하단에 디스플레이됩니다. 제품 섬네일은 전반적인 사이트 디자인과 매치되는 보더 라운드 처리 없이 디자인하였습니다.

  • 제품 화면 레이아웃
  • 홈 화면 모바일 버전
  • 광고 화면 레이아웃

    광고 화면은 제품 섬네일 상단에 배치하였으며, 사용자 편리성을 위해 많은 온라인 사이트의 광고방법과 동일하게 진행하였습니다.

구매 화면

화면 전환이 아닌 팝업 방식으로 디자인하였으며, 볼드 한 선으로 내용을 구분 지었습니다. 메인 색상을 구매 버튼에 사용하여 과한 사용을 제한하였습니다.

구매 화면 앱과 웹 레이아웃

앱과 웹 버전의 레이아웃을 보여주는 이미지로 1,2,3 번호를 통해 특정된 콘텐츠가 어떻게 배치되는지 확인하실 수 있습니다.

  • 구매 화면 웹 버전
  • 구매 상세 화면 앱 버전

구매 상세 화면 (웹 버전)

넓은 화면을 활용하여 3등분 하였고, 선택된 구매 제품 > 받는 사람 > 결재방법 & 환불과 반품 > 총금액과 구매 버튼을 배치하였습니다.

구매 상세 화면 웹 버전

주문 상세 화면

웹 버전은 구매 상세 화면과 동일하게 3등분 하였고, 주문 연도 > 주문 상세 > 배송 현황을 배치하였습니다. 3분할 화면을 통해 많은 스크롤 또는 화면 전환 없이 주문 상세 내역을 확인 가능하게 디자인하였습니다.
